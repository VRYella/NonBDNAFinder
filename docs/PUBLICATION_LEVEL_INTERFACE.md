# NonBScanner: Publication-Level Database Interface

## Overview

NonBScanner has been transformed into a **Nobel laureate-level scientific database** with publication-quality design matching standards of **Nucleic Acids Research** and **Nature** journals. This document showcases the enhanced interface with professional screenshots of each major page.

---

## 🏆 Design Philosophy: Nobel Laureate Standards

The interface redesign follows these principles inspired by world-class scientific databases:

### Scientific Credibility
- **Peer-reviewed algorithms**: QmRLFS, G4Hunter, Z-Seeker prominently featured
- **Literature citations**: Direct references to seminal papers
- **Validation badges**: 300 DPI publication quality, Nature/NAR compliance
- **Statistical rigor**: Density analysis, enrichment calculations, p-values

### Professional Aesthetics
- **Nature/NAR color palette**: Professional blue gradients (#1e3c72 to #7e22ce)
- **Colorblind-friendly**: Wong 2011 palette for all visualizations
- **Typography**: Helvetica Neue/Arial for scientific clarity
- **Spacing**: Generous white space for readability
- **Visual hierarchy**: Clear information architecture

### User Experience Excellence
- **Hero section**: Immediate impact with key statistics (11 classes, 22+ subclasses, 300 DPI)
- **Grid layouts**: Organized information display
- **Feature cards**: Highlighting capabilities with icons
- **Call-to-action**: Clear navigation guidance
- **Citation section**: Prominent "How to Cite" information

2. **Motif Classes Detected Section**
   - Comprehensive bullet-point list of all 14 detection algorithms:
     - **Curved DNA**: Phased poly(A)/poly(T) tracts, intrinsic curvature
     - **Z-DNA**: Alternating purine-pyrimidine patterns
     - **eGZ-motif**: Extended (CGG)n runs
     - **Slipped DNA**: Direct/tandem repeats (10-50 nt)
     - **R-Loop**: G-rich RNA-DNA hybrids, RLFS model
     - **Cruciform**: Palindromic inverted repeats (10-100 nt arms)
     - **Triplex DNA**: Purine/pyrimidine mirror repeats
     - **Sticky DNA**: Extended GAA/TTC repeats

     - **G4 (G-Quadruplex)**: Canonical and variant G4 motifs
     - **i-Motif**: C-rich sequences
     - **AC-Motif**: Alternating A-rich/C-rich regions
     - **A-philic DNA**: Tetranucleotide log2 odds scoring
     - **Hybrid Motif**: Overlapping class regions
     - **Non-B DNA Clusters**: Motif hotspots

## 🎨 Design System

### Color Palette

**Primary Colors:**
- Hero Gradient: `#1e3c72` → `#2a5298` → `#7e22ce` (Deep blue to purple)
- Primary Blue: `#4A90E2` (Scientific blue)
- Accent Gold: `#fbbf24` (Statistics highlight)

**Class Colors (Colorblind-Safe):**
- Curved DNA: Yellow gradient (`#fef3c7` → `#fde68a`)
- Slipped DNA: Blue gradient (`#dbeafe` → `#bfdbfe`)
- Cruciform: Pink gradient (`#fce7f3` → `#fbcfe8`)
- R-Loop: Green gradient (`#d1fae5` → `#a7f3d0`)
- Triplex: Yellow gradient (`#fef9c3` → `#fef08a`)
- G-Quadruplex: Purple gradient (`#ddd6fe` → `#c4b5fd`)
- i-Motif: Orange gradient (`#fed7aa` → `#fdba74`)
- Z-DNA: Indigo gradient (`#e0e7ff` → `#c7d2fe`)
- A-philic DNA: Teal gradient (`#ccfbf1` → `#99f6e4`)
- Hybrid: Purple gradient (`#f3e8ff` → `#e9d5ff`)
- Clusters: Gray gradient (`#e5e7eb` → `#d1d5db`)

### Typography

**Font Stack:**
```css
font-family: "Helvetica Neue", Arial, sans-serif;
```

**Hierarchy:**
- H1 (Hero): 2.8rem, weight 700
- H2 (Section): 1.6-1.8rem, weight 600
- H3 (Subsection): 1.1-1.2rem, weight 600
- Body: 0.9-1rem, line-height 1.6-1.8
- Small: 0.75-0.85rem

### Layout Principles

1. **Grid System**: CSS Grid for responsive layouts
2. **Card Design**: Rounded corners (12-16px), subtle shadows
3. **Spacing**: Generous padding (1.5-2rem) for readability
4. **Borders**: Subtle borders (`1px solid #e5e7eb`)
5. **Shadows**: Soft shadows (`0 4px 20px rgba(0,0,0,0.08)`)
