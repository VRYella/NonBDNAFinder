MATERIALS AND METHODS
================================================================================

Genome Data and Processing
--------------------------------------------------------------------------------
We analyzed 9 complete genome sequences
totaling 34,048,350 base pairs. All sequences
were obtained in FASTA format and processed using the NonBDNAFinder
computational pipeline (version 2025.1).

Non-B DNA Motif Detection
--------------------------------------------------------------------------------
Non-B DNA motif detection was performed using NonBDNAFinder, a comprehensive
computational tool that identifies 11 major structural classes including
G-quadruplexes, Z-DNA, i-motifs, curved DNA, A-philic DNA, cruciform structures,
triplex DNA, R-loops, slipped DNA structures, hybrid motifs, and non-B DNA
clusters. The detection pipeline employs pattern-matching algorithms combined
with thermodynamic scoring models to ensure high sensitivity and specificity.

Each detected motif was assigned a confidence score on a 1-3 scale based on
structural propensity calculations. Motifs were classified into specific
subclasses based on structural features such as loop length, tract arrangement,
and thermodynamic stability. Overlapping motifs within the same subclass were
merged to avoid redundancy, while overlaps between different classes were
flagged as hybrid structures for further analysis.

Statistical Analysis
--------------------------------------------------------------------------------
For each genome, we calculated motif density (motifs per kilobase), genomic
coverage (percentage of sequence occupied by motifs), and class diversity
(Shannon entropy). Cross-genome comparisons were performed using standardized
frequency normalization. Enrichment analysis identified motif classes showing
significantly elevated or depleted frequencies (Z-score > 2.0 or < -2.0)
relative to the cross-genome mean. Pearson correlation analysis was used to
assess relationships between genome features (length, GC content, motif density).

Methodological Considerations
--------------------------------------------------------------------------------
This analysis employed computational prediction methods that, while highly
sensitive and specific, represent potential structural propensities rather
than experimentally confirmed structures. In vivo formation depends on
cellular context including chromatin state, protein binding, and DNA
supercoiling. Nevertheless, computational approaches enable genome-scale
analyses infeasible through experimental methods alone.
