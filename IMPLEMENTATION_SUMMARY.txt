================================================================================
TECHNICAL DETAILS
================================================================================

Job ID Format:
- Length: 10 characters
- Charset: Hexadecimal (0-9, a-f)
- Generation: uuid.uuid4().hex[:10]
- Collision space: 2^40 (1,099,511,627,776 possible IDs)

Storage Structure:
results/
  └── <job_id>/
      ├── results.json      # Complete motif results
      ├── sequences.json    # Original sequences
      └── metadata.json     # Job info and stats

Email Configuration (Optional):
.streamlit/secrets.toml:

SECURITY & PRIVACY
================================================================================

Job ID Security:
  ✓ Non-sequential (UUID-based)
  ✓ Large collision space (2^40)
  ✓ Cryptographically random
  ✓ Not bound to sessions

Data Privacy:
